# 49TI 游戏人格测试

4399 五一趣味测试，探索你的游戏人格。

## 关于

24道题，测出你的游戏灵魂。手游、云游戏全覆盖，看看你是躺平党、氪金怪、社交花蝴蝶，还是肝帝本帝。

本测试基于 49TI 人格体系设计，涵盖四大维度：

- **M 游戏动机**：你为什么玩
- **S 社交风格**：你喜欢怎么和别人一起玩
- **C 竞技心态**：你在游戏里有多上头
- **I 投入方式**：你愿意为游戏花多少时间和钱

共 20 种人格类型，总有一款属于你。

## 使用

直接打开 `index.html` 即可使用，无需服务器。

COLLECTOR.png / CLOUD.png / GOD.png / SHY.png / FOMO.png /
SAGE.png / ALCOHOLIC.png / SHARECARD.png / RECHARGE.png /
RIVAL.png / GUILD_LEADER.png / LOYAL.png / MAINTENANCE.png /
STREAMER.png / WILDLIFE.png

推荐尺寸：400 × 500px，PNG 格式。
